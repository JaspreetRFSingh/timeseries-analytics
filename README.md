# Time-Series Analytics Engine
High-performance time-series data ingestion, storage, and analytics engine.
## Features
- Millions of events per second ingestion
- Multiple aggregation functions (AVG, SUM, MIN, MAX, P50, P95, P99)
- Downsampling with configurable windows
- Columnar storage with Gorilla compression
- Query optimization
## Tech Stack
Java 17, Spring Boot 2.7.13, Apache Kafka
